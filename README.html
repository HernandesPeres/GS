Genomic Selection

Genomic selection (GS) is a strategy used in plant breeding that involves selecting parental individuals, crossing parents, evaluating progeny performance, and selecting top candidates to become new cultivars or parents for hybrid breeding.

It provides breeders an estimate of plant performance before phenotyping, allowing: - Reduced costs - Faster breeding cycles

GS Pipeline

Genomic selection can be divided into four stages:

  1. Training population design
  2. Model building
  3. Prediction
  4. Selection

1. Training Population Design

Historical phenotypic and genotypic data are used to train models.

Example dataset: - 300 genotypes (training) - 200 genotypes (prediction) - 3,000 SNP markers

2. Model Building

Models aim to maximize prediction accuracy.

Common approaches: - RR-BLUP (marker effects) - GBLUP (genomic relationships)
